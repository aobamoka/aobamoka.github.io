大致了解-java1

Last Update:2023-02-15 Reading Time: 2 min Readed:

软件分析课程需要，提前熟悉下java，也不知道学多久太监。

基本结构与类型

类

类 (class) 与方法 (method)。

类名：类名必须以英文字母开头，后接字母，数字和下划线的组合。习惯以大写字母开头。

Java入口程序规定的方法必须是静态方法，方法名必须为main，括号内的参数必须是String数组。

注释：

  • // 单行注释
  • /*...*/ 多行注释
  • /**...*/ 写在类和方法定义处，用于创建文档

基本数据类型

  • 整数类型：byte，short，int，long
  • 浮点数类型：float，double
  • 字符类型：char
  • 布尔类型：boolean

整型

int i = 2147483647;
int i2 = -2147483648;
int i3 = 2_000_000_000; // 加下划线更容易识别
int i4 = 0xff0000; // 十六进制表示的16711680
int i5 = 0b1000000000; // 二进制表示的512
long l = 9000000000000000000L; // long型的结尾需要加L

浮点型

float f1 = 3.14f;
float f2 = 3.14e38f; // 科学计数法表示的3.14x10^38
double d = 1.79e308;

float型必须加f后缀

字符类型

单引号，可表示一个unicode字符

引用类型

除以上三种类型外都是引用类型

常量

final修饰符，只可初始化赋值，不能更改其值

var

使用var代替类型名，让编译器自动推断

字符串

如果用+连接字符串和其他数据类型，会将其他数据类型先自动转型为字符串

从Java 13开始，字符串可以用”””…”””表示多行字符串（Text Blocks）了

流程控制

输出

println是print line的缩写，表示输出并换行。因此，如果输出后不想换行，可以用print()
格式化输出使用System.out.printf()，通过使用占位符%?，printf()可以把后面的参数格式化成指定格式

输入

首先通过import语句导入java.util.Scanner；之后创建Scanner对象并传入System.in；最后，通过Scanner类的nextLine、nextInt等方法来读取输入并传输到String、Int等类型的变量中。

判断引用类型相等

==：用于判断两个引用类型指向的对象是否为同一个
equals方法：用于判断两个引用类型指向的对象值是否相等

switch

java12开始的新语法：->，用于替代case后面的:。
新语法没有穿透效应（所以不用在case末尾添加break），并且还可以通过switch语句直接返回值（而不需要在每个分支给同一个变量赋值）。

如果需要复杂的语句，我们也可以写很多语句，放到{…}里，然后，用yield返回一个值作为switch语句的返回值.
