线段树学习笔记

线段树 学习笔记

1. 背景

树状数组无法满足求给定区间的最大最小值，或是将给定区间全部修改为某一个值的操作。

思想：

  • 分治：某一段区间的问题可分割为更小区间的子问题求解。
  • 完全二叉树：子节点的下标可由父节点计算得到。

以求区间和的线段树为例，代表区间 [l, r] 的结点 tree[i]，其左子结点 tree[2 * i] 表示区间 [l, (l + r) / 2)] 的区间和；其右子结点 tree[2 ∗ i + 1] 表示区间 [(l + r) / 2 + 1, r] 的区间和。

2. 初始化/单点修改/区间查询

一般初始化为4 * n大小

2.1 单点修改

总体为后序遍历。

将要修改的值沿树一路传递下去，直到到达叶子节点实现单点修改。

在返回值时，需要更新父节点的值为两个子节点的值的和。

2.2 区间查询

如果目前找到的线段树表示的区间是待查询区间的子集，则直接返回当前节点的值。

否则，将当前表示的区间折半分为两部分，分别对左区间与右区间递归查询，最后将递归查询的结果汇总。

求区间和/区间最值时，只需要将父节点的更新行为更改成求和/求最值即可。

同样是[LeetCode 307. 区间和检索-数组可修改](class NumArray { public: vector tree; int n; void pushUp(int i) { tree[i] = tree[i * 2] + tree[i * 2 + 1]; } NumArray(vector& nums) { n = nums.size(); tree = vector(4 * n, 0); for (int i = 0; i < n; i++) { update(i, nums[i]); } } void update2(int index, int val, int l, int r, int i) { if (l == r) { tree[i] = val; return; } int mid = l + (r - l) / 2; if (mid >= index) update2(index, val, l, mid, i * 2); else update2(index, val, mid + 1, r, i * 2 + 1); pushUp(i); } void update(int index, int val) { update2(index, val, 0, n - 1, 1); } int sumRange2(int left, int right, int l, int r, int i) { if (left <= l && r <= right) return tree[i]; int mid = l + (r - l) / 2, sum = 0; if (left <= mid) sum += sumRange2(left, right, l, mid, i * 2); if (right > mid) sum += sumRange2(left, right, mid + 1, r, i * 2 + 1); return sum; } int sumRange(int left, int right) { return sumRange2(left, right, 0, n - 1, 1); } }; /** * Your NumArray object will be instantiated and called as such: * NumArray* obj = new NumArray(nums); * obj->update(index,val); * int param_2 = obj->sumRange(left,right); */)

3. 区间修改

引入与线段树数组同等大小的懒惰标记数组。

要修改某一区间时，按照区间查询那样分治到待查询区间的子集。

按照该子集的大小修改当前线段树节点的值，并在该节点的子节点上打上懒惰标记，表示需要修改的值，暂时不作修改。

在下次遍历到含有懒惰标记的节点时，同样修改当前节点的值并向下传递懒惰标记。

4. 离散化

特征：数据范围过大，但是存在很多重复的点/无意义的点。

做法：先遍历数据去重，并根据map建立映射。

5. 动态开点

结点数组法：将左右节点的下标记录在父节点中，而不是直接计算。

结点引用法：直接按照二叉树那样建树。
